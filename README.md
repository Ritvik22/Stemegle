# Stemegle

Fast, competitive STEM battles for curious minds. Players can enter as a named guest, wait for an available opponent, answer timed STEM questions, and compete for score, XP, streaks, and universal rank.

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite.

## Production build

```bash
npm run build
```

## Current scope

This repository contains the responsive React front end and a playable simulated match flow. Real accounts, player presence, matchmaking, synchronized battles, and persistent rankings require a multiplayer backend.
